# TechWave Pro - IT Software Services Theme

## Description
Modern, professional IT Software Services theme with vibrant orange and deep blue color scheme. Features advanced button designs, gradient effects, and enterprise-ready UI components.

## Features
- **Advanced Button Styles**: Gradient buttons with hover effects, shine animations, and shadows
- **Glassmorphism Effects**: Modern glass card designs with backdrop blur
- **Gradient Text**: Eye-catching gradient text for headings and CTAs
- **Enhanced Cards**: Hover animations, border gradients, and depth effects
- **Modern Navigation**: Sticky header with underline hover effects
- **Responsive Design**: Mobile-first approach with breakpoint optimization
- **Custom Scrollbar**: Styled scrollbar matching theme colors

## Color Scheme
- **Primary Orange**: #ff6b35 (vibrant, energetic)
- **Secondary Blue**: #1e40af (professional, trustworthy)
- **Accent Gold**: #fbbf24 (premium, highlighting)
- **Dark**: #1f2937 (footer, contrast)
- **Light**: #f9fafb (backgrounds)

## Typography
- **Primary Font**: Inter (body text, clean and readable)
- **Headings Font**: Poppins (modern, bold headlines)
- **Monospace**: Fira Code (code snippets)

## Installation
1. Upload the ZIP file via Admin Panel → Templates → Upload Template
2. Extract and activate the template
3. Clear cache if needed
4. Customize colors via template settings
