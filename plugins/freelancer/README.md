# Freelancer Marketplace Plugin

## Descriere
Plugin complet de marketplace freelancing pentru Carphatian CMS, similar cu Fiverr sau Upwork.

## Funcționalități
- ✅ Profiluri complete pentru freelanceri
- ✅ Sistem de Gig-uri (servicii oferite)
- ✅ Management comenzi cu statusuri multiple
- ✅ Sistem de mesagerie integrat
- ✅ Tracking câștiguri pentru freelanceri
- ✅ Rating și review-uri
- ✅ Menu frontend automat

## Instalare
1. Accesează `/admin/plugins` în panoul admin
2. Click pe butonul "Upload Plugin ZIP"
3. Selectează fișierul `freelancer-plugin.zip`
4. Click "Activează" după instalare
5. Refresh pagina pentru a vedea meniurile

## Admin Menus (după activare)
- **Comenzi Freelancer** - Gestionează toate comenzile
- **Profiluri Freelancer** - Administrează profilurile
- **Gig-uri** - Managementul serviciilor oferite
