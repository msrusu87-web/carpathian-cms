# Screenshots

This directory contains screenshots of Carpathian CMS.

## How to Add Screenshots

1. Take screenshots of:
   - Admin dashboard
   - AI content generator
   - Product management
   - Page builder
   - Multilingual features
   - Frontend homepage

2. Save them with these names:
   - `admin-dashboard.png`
   - `ai-generator.png`
   - `product-management.png`
   - `page-builder.png`
   - `multilingual.png`
   - `frontend-home.png`

3. Recommended size: 1920x1080 or 1280x720
4. Format: PNG or JPG
