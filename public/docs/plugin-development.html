Plugin Development Guide

Learn how to create and upload plugins for Carpathian CMS

Upload Plugin in Admin

  1. 1 Go to Admin → CMS → Plugins
  2. 2 Click "Upload Plugin Package" button (green, top of table)
  3. 3 Select your ZIP file containing package.json and plugin.php
  4. 4 After upload, click "Activate" to enable the plugin

Package Structure

Your plugin ZIP must contain at minimum:

  • package.json - Manifest with name, slug, version
  • plugin.php - Your plugin code
