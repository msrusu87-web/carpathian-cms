Template Development Guide

Learn how to create and upload templates for Carpathian CMS

Upload Template in Admin

  1. 1 Go to Admin → CMS → Templates
  2. 2 Click "Upload Template Package" button (green, top of table)
  3. 3 Select your ZIP file with package.json, style.css, and layouts/
  4. 4 After upload, activate it and optionally set as default template

Package Structure

Your template ZIP must contain:

  • package.json - Manifest with name, slug, version, color_scheme
  • style.css - Your template CSS
  • layouts/ - Directory with JSON layout files
